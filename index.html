About Me

I am a Data Scientist & AI Developer with strong hands-on experience in building real-world machine learning and deep learning models, supported by academic research and impactful projects. My background spans predictive modeling, natural language processing (NLP), computer vision, image processing, and data analysis, with proficiency in tools and languages such as Python, SQL, TensorFlow, and Scikit-Learn. Projects

  • Heart Disease Prediction – GitHub
    Developed a heart disease prediction model using RF, XGBoost, CatBoost, SVM, and 1D CNN. Achieved 96.36% accuracy with 1D CNN by applying advanced feature engineering and PCA. Boosted model performance by 7% through interaction-driven feature analysis and identified key health risk indicators.
  • Hybrid Deep Ensemble for Sentiment Analysis – GitHub
    Built a hybrid sentiment analysis model combining BERT-based augmentation, transformer embeddings, ensemble learners, and a CNN-BiGRU-Attention network. Achieved 91.15% accuracy on Amazon reviews using stacking and Optuna-based hyperparameter tuning.
  • Brain Tumor Classification – GitHub
    Developed a CNN pipeline leveraging transfer learning with VGG16, ResNet50, InceptionV3, and EfficientNetB0. Applied advanced image augmentation using Albumentations and achieved 97% accuracy through a stacking ensemble with logistic regression.
  • Airbnb-price-prediction – GitHub
    This project builds a powerful Airbnb price prediction model using deep learning (Keras ANN) and ensemble methods like XGBoost, CatBoost, and Gradient Boosting. Strong feature engineering, log transformation, and a stacking ensemble achieved high accuracy with an R² up to 0.999.
